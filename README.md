# Requirement
도로명 주소 검색
1) 도로명은 null이 되면 안된다(필수값)
2) 건물 번호에는 본번과 부번이 있다
3) 본번만 입력되었을 시 건물번호가 일치하는 도로명 주소가 검색된다
4) 부번이 함께 입력되었다면, 본번과 부번이 모두 일치하는 도로명 주소를 검색한다
